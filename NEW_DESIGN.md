# DronWay - новый дизайн на весь сайт

## Что сделано
Весь многостраничник переведён на дизайн-систему прототипа (Space Grotesk + Inter,
светлая тема, фиксированная навигация, фирменные шапка/подвал).

Ключ: `src/styles/global.css` переписан в стиль прототипа, но СОХРАНЕНЫ прежние
имена классов (`.sec`, `.card`, `.grid`, `.spec`, `.chips`, `.flist`, `.ctaband`,
`.specband`, `.dtable` и т.д.) и оставлены legacy-переменные (`--blue`, `--muted`...).
Поэтому существующие страницы переоделись без переписывания разметки.

- `src/layouts/Base.astro` - новая шапка/подвал (общие для всех внутренних страниц).
- `src/components/PageHead.astro` - светлая страница-шапка с хлебными крошками.
- `src/layouts/Site.astro` + `src/pages/index.astro` - бесшовная главная (герой с DW-150,
  скролл-секции с реальными бортами, тёмная «Выше нормы», схема «4 / 11 шагов»,
  «Шесть причин», линейка, CTA). Изолирована, чтобы её фуллскрин-секции не конфликтовали
  с общим словарём классов.
- `public/drones/*` - борта без фона (WebP x3 + PNG), подключены на главной через srcset.
- Длинные тире (—) убраны во всём `src`.

Страницы на новом дизайне (через Base + global.css): каталог и карточки бортов,
услуги, применение, виды дронов, этапы, дрон-как-сервис, обучение, партнёрам, о нас, контакты.

## Запуск
```
npm install
npm run dev      # просмотр
npm run build    # сборка в ./dist
```

## Дальше
Пришлёшь материал - дополним внутренние страницы контентом и при желании добавим
фуллскрин-секции с фото бортов (как на главной) на ключевые страницы (каталог, применение).
